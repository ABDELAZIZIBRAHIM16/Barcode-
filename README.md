# تطبيق ماسح باركود محل أطفال (Kids Store Scanner)

هذا التطبيق مصمم لمساعدة موظفي المبيعات في محلات مستلزمات الأطفال على معرفة أسعار المنتجات بسرعة من خلال مسح الباركود.

## الميزات الرئيسية
*   **مسح سريع للباركود:** يستخدم Google ML Kit لقراءة الباركود بدقة عالية.
*   **دعم تنسيق 000-000:** يفصل الكود إلى (رقم المجموعة) و (رقم الصنف) للمطابقة مع قاعدة البيانات.
*   **العمل دون اتصال (Offline):** يتم تخزين البيانات محليًا باستخدام Room Database.
*   **استيراد من Excel:** إمكانية تحديث الأسعار والمنتجات عبر ملف Excel.
*   **بناء تلقائي (GitHub Actions):** مجهز بملف تكوين لبناء الـ APK تلقائيًا عند الرفع على GitHub.

## كيفية الرفع على GitHub والحصول على الـ APK:
1. قم بإنشاء مستودع (Repository) جديد على GitHub باسم `KidsStoreScanner`.
2. قم برفع محتويات هذا المجلد إلى المستودع.
3. اذهب إلى تبويب **Actions** في صفحة المستودع على GitHub.
4. ستجد عملية بناء (Workflow) بدأت تلقائيًا باسم "Android CI/CD".
5. عند انتهاء البناء (تظهر علامة صح خضراء)، انقر على العملية، ثم انزل للأسفل لتجد قسم **Artifacts**.
6. ستجد ملف `app-debug` متاحًا للتحميل، وهو يحتوي على ملف الـ APK الجاهز للتثبيت.

## التقنيات المستخدمة
*   **اللغة:** Kotlin
*   **واجهة المستخدم:** Jetpack Compose
*   **قاعدة البيانات:** Room Persistence Library
*   **مسح الباركود:** Google ML Kit Vision
*   **الكاميرا:** CameraX
